# google_colab_ML

This repository contains my machine learning "playground" notebooks built in Google Colab.

## Notebooks

- **IMDB Sentiment Analysis**  
  A TensorFlow/Keras notebook that trains a movie review sentiment classifier on the IMDB dataset using:
  - `tf.data` input pipelines
  - `TextVectorization` for tokenization
  - An `Embedding` layer to learn word vectors
  - A bidirectional LSTM encoder and dense classification head  
  The notebook trains, evaluates, and runs predictions on custom text, all using Colab’s free GPU.

More notebooks will be added here as I experiment with different ML models and datasets.
